# mmstyles

Latex styles for paper writing in Multimedia Lab.

## Abbreviations

``\ie``
``\eg``
``\wrt``

## Operators

``\argmin``
``\argmax``

## Letters in special fonts

#### Lower-case in bold (for vectors)

- ``\va``, ``\vb``, ..., ``\vz``
- ``\vzero``, ``\vone``
- ``\valpha``, ``\vbeta``, ... ``\vomega``

#### Upper-case in bold (for matrices)

- ``\mA``, ``\mB``, ..., ``\mZ``
- ``\mGamma``, ``\mDelta``, ..., ``\mOmega``

### Upper-case in calligraphic fonts

- ``\cA``, ..., ``\cZ``
